# 📗 Linear Data Structures in C++

This repository contains implementations of core **Linear Data Structures** in C++ built from scratch without using built-in libraries.

## 📂 Topics Covered
- Singly, Doubly & Circular Linked Lists  
- Stacks (Array, Vector, Linked List)  
- Queues & Priority Queues  
- Menu-driven real-world systems

These programs focus on dynamic memory management, pointer usage, and applying data structures to practical problems.

## 🛠️ Tech Stack
Language: **C++**
